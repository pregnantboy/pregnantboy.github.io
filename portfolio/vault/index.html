Description

Vault is a password manager for teams, packed into a chrome extension. It uses Cloud Firestore by Firebase for storage and permissions, Google OAuth 2.0 for authentication and based on Material Design for front end.

Features:

  • Real-time account sharing/syncing
  • Smart autofilling of login forms
  • Permission control for each account
  • Strong encryption

Lunch Time is an add-on where a team member can initiate a lunch session. Other team members will receive a desktop notification and be given an option to join in. When lunch time is up, users will be prompted.
If they refuse to make their way to the door, a barrage of popups will appear on their screen.
