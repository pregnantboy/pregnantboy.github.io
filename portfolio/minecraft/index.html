Description

This project aims to introduce the basics of AI to youths through simplified programming by incorporating fuzzy logic in Minecraft.

The objectives are:

  1. To allow users to learn how AI is employed in Non-Player Characters (NPCs) of video games through hands-on programming of their own AI agents.
  2. To spark interest in programming and AI in young users through video games.
  3. To encourage creative and critical thinking in users when developing their own AI agents.
  4. To increase the number of possible ways that Minecraft can be played to provide a fresh gameplay experience for current players.

This project employs the characteristics of two AI techniques by incorporating fuzzy logic into behaviour trees. This is done by using a Fuzzy Inference System (FIS) as the main selector node. The FIS will compute the degree of support for each rule and then rank the subtrees from highest degree to lowest and attempt to execute them in this preference. Futhermore, fuzzy logic allows the use of natural language in the creation of rules. Instead of specifying crisp values like true, false or any numerical value, descriptive words like low, moderate or high can be used. This allows the rules to be more readable and easily interpreted by younger users.
